# stock-ticker

## 顯示內容

在狀態列底部以獨立一行、由左至右捲動的台股報價跑馬燈 — 指數加上指定個股，含股價與漲跌百分比，依台灣慣例上漲標紅、下跌標綠（`TWII 44,733 +2.5% · 廣達 339.00 +9.9%`）。

## 資料來源

- **TWSE MIS**（`mis.twse.com.tw`）— 免金鑰的公開報價端點。每次抓取以單一請求帶入多檔代碼。`render` 只讀取本地快取；`fetch`（每 60 秒）負責更新快取。

## 設定

| 鍵 | 型別 | 預設值 | 意義 |
|-----|------|---------|---------|
| `symbols` | array | `["t00","0050","2353","2382","3231"]` | TWSE 代碼。`t00` = 加權指數，4 碼 = 上市代碼，`otc:<code>` = 上櫃。 |

## 需求

- PATH 上要有 `python3`
- 可連線至 `mis.twse.com.tw`（僅在背景抓取時）

## 安全性說明

- `render` 完全不碰網路 — 只讀取已快取的報價。
- 無任何密鑰、無需驗證、不會寫入元件 state 目錄以外的位置。

## 輸出範例

```
 TWII 44,733 +2.5% · 元大台灣50 105.40 +4.9% · 宏碁 35.20 +10.0% · 廣達 339.00 +9.9% · 緯創 158.50 +9.7%
```
