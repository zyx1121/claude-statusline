# stock-ticker

## 显示内容

在状态栏底部以独立一行、从左到右滚动的台股报价跑马灯 — 指数加上指定个股，含股价与涨跌百分比，按台湾惯例上涨标红、下跌标绿（`TWII 44,733 +2.5% · 廣達 339.00 +9.9%`）。

## 数据来源

- **TWSE MIS**（`mis.twse.com.tw`）— 免密钥的公开报价端点。每次抓取以单个请求带入多个代码。`render` 只读取本地缓存；`fetch`（每 60 秒）负责刷新缓存。

## 配置

| 键 | 类型 | 默认值 | 含义 |
|-----|------|---------|---------|
| `symbols` | array | `["t00","0050","2353","2382","3231"]` | TWSE 代码。`t00` = 加权指数，4 位 = 上市代码，`otc:<code>` = 上柜。 |

## 依赖

- PATH 上需有 `python3`
- 可访问 `mis.twse.com.tw`（仅在后台抓取时）

## 安全说明

- `render` 完全不访问网络 — 只读取已缓存的报价。
- 无任何密钥、无需认证、不会写入组件 state 目录以外的位置。

## 输出示例

```
 TWII 44,733 +2.5% · 元大台灣50 105.40 +4.9% · 宏碁 35.20 +10.0% · 廣達 339.00 +9.9% · 緯創 158.50 +9.7%
```
